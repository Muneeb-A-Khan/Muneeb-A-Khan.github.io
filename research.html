Research Projects

Post-Processing Methods for Artifact Removal Using Machine Learning

Funding: Google Korea (2024-Present) | Affiliation: Google Korea & Pi Lab

Investigating machine learning techniques for suppressing noise, ghosting, and compression artifacts in diverse imaging modalities. Focus on optimizing performance and designing deployment strategies for mobile and resource-constrained environments.

Multi-Modal Framework for Medical Condition Classification and Detection

Affiliation: Pi Lab

Developing a multi-class detection system using CNNs for conditions like brain tumors, COVID-19, pneumonia, and lung opacities. Addressing challenges of ambiguous boundaries in medical images and optimizing models for clinical environments with explainability.

Transformer-Enhanced Diffusion Models for Image Segmentation and Super-Resolution

Affiliation: Pi Lab

Exploring diffusion techniques enhanced with transformers for improved image segmentation and resolution enhancement. Integrating text-guided diffusion processes to increase precision and validating performance across various imaging modalities.

Objective Quality Metrics for Ghosting Artifacts in Video and HDR Images

Funding: Google Korea (2023-2024) | Affiliation: Pi Lab

Built a large-scale dataset (M-GAID) with over 2,500 real images and 37,000 annotated patches for spatial/temporal artifact detection. Designed data pipelines and trained SOTA deep learning models for ghosting artifact detection. Published at WACVW 2025.

Multi-Scale Attention Model for Low-Light Image Enhancement

Affiliation: Pi Lab (2024)

Designed a lightweight model (12M parameters) achieving high performance metrics on LoL datasets using GCP A100. Integrated attention mechanisms and transfer learning, implementing an adaptive enhancement pipeline balancing quality and efficiency.

Brain Tumor Detection Using Magnetic Resonance Imaging

Affiliation: Pi Lab (2022-2024)

Developed a lightweight convolutional block architecture achieving high accuracy (99.51% mAP) and fast inference (17.2 ms). Performed tumor segmentation, classification, and anomaly detection, significantly reducing false positives using XAI techniques (Grad-CAM, LIME).

FireXplainNet: Interpretable Wildfire Detection System

Affiliation: Pi Lab (2023-2024)

Created a lightweight (5.3M parameters) interpretable CNN for early wildfire detection. Applied Grad-CAM for explainability in high-risk scenarios. Achieved high accuracy and received the Best Paper Award at KCC 2023.

YOLO-Based Real-Time Object Detection and Tracking System

Affiliation: Pi Lab (2022-2023)

Built a real-time multi-object detection and tracking system using YOLOV5/YOLO-NAS and DeepSORT/ByteTrack. Achieved mAP > 92% and 30+ FPS on COCO/MOT datasets, leveraging GPU parallelization.

Traffic Sign Recognition with Advanced Neural Network Techniques

Affiliation: Pi Lab (2021-2023)

Developed an interpretable CNN (2.6M parameters) achieving 98.4% accuracy and 74.34 ms inference on GTSRB/ITSD/PTSD datasets. Streamlined development with MLflow and optimized GPU acceleration. Accepted at CVPRw 2025.
